# RealtyDeploy

This is a Next.js 13.5.11 real estate project using Contentlayer for MDX/blog support, designed for deployment on Vercel.

- Compatible with Next.js 13.5.11 and next-contentlayer@0.3.4
- For deployment issues, ensure your package.json specifies "next": "13.5.11"
- See package.json for all dependencies

## Local Development

```bash
npm install
npm run dev
```

## Deployment

Push to the main branch to trigger a Vercel deployment.
